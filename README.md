# Handwritten-Digit-Recognition

Hi All,

I am developing this project for my own learning purpose. The main goal of this project is to understand how AI works and learn how to develop one from scratch.

In this project we are going to develop a small AI which will be able to recognise handwritten digits.

Now, there are several way's one can implement this such as K-Nearest Neighbors (KNN), Support Vector Machines (SVM), Artificial Neural Networks (ANNs), etc. But we are going to use the Convolutional Neural Networks (CNNs) which is best suited and recommended for this purpose.

We will be using MNIST dataset in csv format as input for training as well as testing. This csv includes the first column as label and first row as position and each cell contains the pixel value in back and white in range of 0 to 255.
